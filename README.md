### fonctionnement
1. exec le fichier main.py
2. dire "Jarvis" pour declancher la reconnaissance vocal
3. dire ce que l'on veux (pour l'instant "Stop toi" ou "Parle")
4. Ecouter
    1. boucle etape 2
